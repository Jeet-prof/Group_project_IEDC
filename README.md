# Text Summarization using EasyOCR

## Description
This project extracts text from images using EasyOCR, then generates a summarized version of the extracted text.

## Features
- Extract text from images
- Generate text summaries
- Simple Python implementation

## Requirements
- Python 3.10+
- EasyOCR
- OpenCV
- Pillow
- Transformers
- Torch

## How to Run
1. Create a main folder and within that folder create another 2 folders: Images and Output; and 5 files: Main.py, preprocess.py, ocr.py, cleaner.py and segment.py 

2. Copy and paste the python codes according to the files. 

3. Install the required libraries:
   pip install easyocr opencv-python numpy pillow

4. Put an image in the image folder. Any clear image with English text is fine for testing.

5. Run the program:
   python main.py
